INTRODUCTION
------------

This repo allows us to maintain master copies of all non-contrib modules
created by Classic Graphics dev.

Head is currently 7.x.

CONTENTS
------------

fusion_pro - This module provides integration to the Fusion Pro server.

fp_template - This module provides a non-programmatic way to generate 
  template previews from the Fusion Pro server.

fp_template_configs - This module provides the necessary template
  configurations for interacting with the fusion_pro module.
