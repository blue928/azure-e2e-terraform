INTRODUCTION
------------

Fusion Pro provides an API for use by other modules to interact with the
Fusion Pro server.

INSTALLATION
------------

1. Copy this fusion_pro/ directory to your sites/SITENAME/modules directory. 

2. Configure user permissions at admin/user/permissions.

3. Go to admin/settings/config/fusion_pro to add configurations.
